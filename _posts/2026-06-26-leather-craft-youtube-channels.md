Curated list of YouTube channels sharing leather craft tutorials, tips, patterns, and full project build-alongs — from absolute beginner to advanced. Verified via Tavily web search, Feedspot rankings, Leatherworker.net community, and direct YouTube @handle checks — June 2026.

| YouTube | What You'll Learn |
|---------|-------------------|
| [**Corter Leather**](https://youtube.com/@Corter) | Budget-friendly beginner tutorials, free patterns, project builds — Eric's "How to Start Leather Crafting" is a classic |
| [**Weaver Leather Supply**](https://youtube.com/@WeaverLeatherSupply) | Chuck Dorsett's *Leather Element* series: tools, techniques, and projects; one of the largest leathercraft catalogs on YouTube |
| [**Tandy Leather**](https://youtube.com/@TandyLeatherCo) | Supplier-backed tutorials from beginner to pro; tool proficiency, starter projects, workshops |
| [**Dark Horse Workshop**](https://youtube.com/@DarkHorseWorkshop) | Traditional leathercraft methods, detailed SCA armor patterns, excellent beginner project walkthroughs |
| [**Leodis Leather**](https://youtube.com/@Leodis.Leather) | Ian Atkinson's huge library — basic wallets to advanced bags; techniques, workspace tours, tool reviews |
| [**Fischer Workshops**](https://youtube.com/@FischerWorkshops) | Design-forward projects, tracing skills, unique everyday carries with standout style |
| [**J.H. Leather**](https://youtube.com/@JHLeather) | Extensive free online tutorials covering skills, techniques, and full projects; *Leather Craft 101* series for newcomers |
| [**Don Gonzales**](https://youtube.com/@DonGonzalesMaker) | **106K subscribers** — saddle making, western floral design, tooling instruction from a master |
| [**Bruce Cheaney**](https://youtube.com/@cheaneysaddles) | Saddlemaker sharing shop tips, leather craft tutorials, and how-to's from decades of experience |
| [**Tony See Patterns**](https://youtube.com/@TonySeePatterns) | aka DieselPunkRo — beginner-friendly patterns, step-by-step DIY builds, free PDF downloads |
| [**Armitage Leather**](https://youtube.com/@ArmitageLeather) | Nigel Armitage's advanced stitching techniques — mesmerizing close-ups, saddle stitch mastery |
| [**MAKESUPPLY**](https://youtube.com/@MAKESUPPLY) | Modern leathercraft templates, tutorials, DIY kits, product reviews, and small-batch supplies |
| [**Stock & Barrel Co**](https://youtube.com/@StockAndBarrel) | Parker's leather goods builds — clean production style, material & supply insights |
| [**The Leathercraft Academy**](https://youtube.com/@TheLeathercraftAcademy) | Hand-made-only leather goods: stitching, molding, pricking, skiving — entirely dedicated to leatherwork |
| [**Little King Goods**](https://youtube.com/@LittleKingGoods) | Clean modern project builds, premium DIY leathercraft kits, minimalist aesthetic |
| [**Road Agent Leather**](https://youtube.com/@RoadAgentLeather) | Holster making, leather tooling, pattern creation — focused on functional carry goods |
| [**Claridge Leather**](https://youtube.com/@claridgeleather) | Building skills through progressively challenging projects, clear explanations |
| [**Castillo Heritage Crafts**](https://youtube.com/@CastilloHeritageCrafts) | Free scrap leather patterns, 5–6 project roundups, great for beginners on a budget |
| [**Leather Pattern TS**](https://youtube.com/@LeatherPatternTS) | Unique leather patterns with step-by-step craft tutorials — keychains, organizers, wallets |
| [**Gary Ipip**](https://youtube.com/@garyipip) | Starter-friendly projects: camera straps, watch bands, everyday leather accessories |
| [**SkillTree**](https://youtube.com/@SkillTree) | Engaging leather craft tutorials with cosplay and prop-making crossover, great for creative makers |
| [**Harry Rogers**](https://youtube.com/@harryrogers) | Leatherwork hints, tips, hacks, and clean project builds |
| [**Ozark Leather Co**](https://youtube.com/@OzarkLeatherCo) | "Start Leathercraft for $50" — budget-focused beginner guides, minimal tool approach |
| [**Prince Armory**](https://youtube.com/@PrinceArmory) | Fantasy armor leatherwork, advanced tooling, carving, and design — stunning artistry |
| [**23plus**](https://youtube.com/@joemeling3216) | Joe Meling — tooling tutorials and practical demonstrations to shorten the learning curve |

## Summary

- **Absolute beginners start here**: Corter Leather, Weaver Leather Supply, J.H. Leather, Ozark Leather Co, Castillo Heritage Crafts
- **Tooling & carving masters**: Don Gonzales (106K), Bruce Cheaney, Prince Armory, 23plus
- **Pattern-heavy channels**: Tony See Patterns (DieselPunkRo), Leather Pattern TS, MAKESUPPLY, Dark Horse Workshop
- **Advanced technique**: Armitage Leather (Nigel Armitage), The Leathercraft Academy, Leodis Leather
- **Sources verified**: Feedspot Top Leather YouTube Channels, Leatherworker.net community list, ManMadeDIY roundup, LearnLeathercraft directory, r/Leathercraft Reddit recommendations
